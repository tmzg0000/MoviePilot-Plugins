# NovaHD任务领取

MoviePilot 插件，每月定时为 [NovaHD](https://pt.novahd.top) 领取任务，领取后推送飞书消息通知。

## 功能

- 每月定时执行NovaHD任务领取（支持自定义Cron表达式）
- 支持自定义任务ID（exam_id），默认领取任务ID为3
- 自动从MoviePilot站点管理中获取Cookie（支持一键获取）
- 领取结果记录到详情页，最多保留30条历史记录
- 飞书推送任务领取结果（成功/失败）

## 配置说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用插件 | 否 | 开关插件 |
| 发送通知 | 是 | 是否发送飞书推送通知 |
| 立即运行一次 | 否 | 保存配置后立即执行一次任务领取 |
| 任务ID (exam_id) | 3 | NovaHD的任务ID |
| 执行周期 | `0 0 1 * *` | Cron 表达式，默认每月1日0点执行 |
| Cookie | - | NovaHD的完整Cookie，可点击"获取Cookie"按钮从站点管理自动读取 |

## 使用方式

1. 在 MoviePilot **站点管理**中添加 `pt.novahd.top` 站点并配置好Cookie
2. 在插件市场安装本插件
3. 点击"获取Cookie"按钮自动填入Cookie，或手动填写
4. 启用插件，保存配置
5. 插件会在每月1日0点自动领取任务，并通过飞书推送结果通知

也可通过远程命令 `/nhdclaim` 或 API `POST /run` 手动触发。

## 数据存储

插件数据保存在 MoviePilot 数据库中：
- `claim_records`：任务领取历史记录（最多保留30条）
