# 躺平自动抽奖助手

MoviePilot 插件，支持 [躺平站点](https://www.tangpt.top) 的自动抽奖和老虎机游戏。

## 功能

### 自动抽奖
- 定时执行躺平站点每日抽奖，支持自定义抽奖次数
- 每次抽奖结果记录到日志
- 飞书推送汇总本次运行所有奖励结果
- 奖品解析支持多种 API 响应格式

### 老虎机
- 定时执行老虎机免费旋转和付费旋转
- 期望值（EV）分析：基于真实赔率数据和奖池计算每次旋转的期望收益
- 仅期望盈利时抽：EV 为负时自动跳过付费旋转，避免亏损
- 支持 Jackpot（大奖）检测与通知
- 自动刷新 spin_token，确保每次旋转使用有效的令牌
- 飞书推送汇总老虎机结果，包含 EV 明细

## 配置说明

### 全局设置
| 配置项 | 说明 |
|--------|------|
| 启用插件 | 开关插件 |
| 启用飞书通知 | 是否发送飞书推送通知 |
| 立即运行一次 | 保存配置后立即执行一次任务 |

### 抽奖设置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 抽奖次数 | 1000 | 每次运行的抽奖次数 |
| 执行周期 | `0 8 */7 * *` | Cron 表达式，默认每 7 天早上 8 点执行 |

### 老虎机设置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 启用老虎机 | 否 | 开关老虎机功能 |
| 最多付费次数 | 50 | 每次运行的最大付费旋转次数 |
| 仅期望盈利时抽 | 是 | EV 为负时跳过付费旋转 |
| 执行周期 | `0 8 * * *` | Cron 表达式，默认每天早上 8 点执行 |

### Cookie
- 站点 Cookie（`c_secure_pass`），用于 API 认证

## EV 计算说明

老虎机期望值基于站点 API 返回的实时赔率数据计算：

- **底注（base_cost）**：每次旋转的基础消耗
- **各奖级概率**：从 `prize_rows` 读取每个奖级的命中概率
- **各奖级派彩**：概率 × 倍率 × 底注
- **基础 EV**：各奖级期望派彩之和 − 底注
- **Jackpot 期望**：历史 Jackpot 概率 × 当前奖池金额
- **综合 EV**：基础 EV + Jackpot 期望
- **RTP（Return To Player）**：期望派彩 / 底注 × 100%

当综合 EV < 0 时，长期玩会导致亏损；EV > 0 时玩家具有数学优势。

## 数据存储

插件数据保存在 MoviePilot 数据库中：
- `slot_records`：老虎机历史记录（最多保留 50 条）
- `lottery_records`：抽奖历史记录（最多保留 50 条）
